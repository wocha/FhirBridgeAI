## Migration / Rollback / Verification

- Apply migrations with `python -m fhirbridge.scripts.migrate_db` before starting runtime services.
- Verify schema version before rollout completion; runtime startup rejects mismatches by design.
- For rollback and repair steps, use `docs/runbooks/database-migration-runbook.md`.
- Reproduce the documented hardening evidence with `python scripts/security/run_hardening_evidence.py`.
